# SHOPEZ – E-Commerce Application

SHOPEZ is a full-stack e-commerce web application developed using the MERN stack.
The project allows users to browse products, add them to cart, and place orders.
Admins can manage products, users, and orders through a secure dashboard.

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features
- User authentication (Login / Register)
- Product listing and search
- Cart and checkout
- Order management
- Admin dashboard

## Project Architecture
- Frontend: React-based SPA
- Backend: REST APIs using Express
- Database: MongoDB
- Authentication: JWT

## Deployment
- Frontend: Netlify
- Backend: Render
